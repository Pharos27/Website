Dawnyx Band

Shape your nights. Choose your mornings.

Overview Features Materials

This project aims to build a custom smartwatch designed specifically to improve waking up through a smarter, gentler alarm system. The device uses a heart-rate sensor and gyro sensor to detect lighter sleep within a set wake-up window, allowing it to trigger the alarm at an optimal moment. Instead of loud sound, it uses a sequence of thermal stimulation on the wrist followed by escalating vibrations to wake the user gently without disturbing others. The watch will include a microcontroller, custom firmware, and a 3D-printed enclosure, creating a practical, fully functioning wearable tailored for sleep-focused use.
